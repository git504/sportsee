Installation

Clone project

git clone https://github.com/git504/sportsee.git

🚀 Front-End launching

cd Client && npm install && run start

🔌 localhost:3000

🚀 then... Back-End launching

cd ../Server && nvm install 12.18 && npm install && nvm use 12.18 && npm run start

🔌 localhost:1234
